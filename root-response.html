This domain's endpoints are proxied by Envoy proxy <3. Every service is containerized by Docker.

Time (Golang microservice)

There are two endpoints - one for getting default current time, another for getting current time in the specified country/language format.

GET /time/

Gets current time in default Golang format. Nothing more, nothing less. If you don't know what time is it, just ask that endpoint!

Examples:

  • /time/

GET /time/[lang-COUNTRY]/

Gets current time in the format usually used in the specified country. Are you in French? In USA? Try these endpoints and never get lost!

Examples:

  • /time/en-US/
  • /time/cs-CZ/
  • /time/fr-FR/
  • /time/en-GB/

Greeting (Node.js microservice)

There is only one endpoint which retrieve a greeting in randomly picked language. It also tells you if that greeting is formal or not.

GET /greeting/

Gets a greeting from randomly picked language. Wanna say hello to your boss or hi to your new friend? You will be the king with this endpoint.

Examples:

  • /greeting/

Calc (Go microservice)

You can do basic arithmetic calculations by this online service. There are four endpoints for each of these operations: +, -, *, /.

GET /calc/[op]/?nums=[num],[num],[num]...

Endpoint for calculating [op] of the given numbers in nums query parameter. The allowed [op]s are add, subtract, multiply and divide.

Examples:

  • /calc/add/?nums=1,2,-3,4,5
  • /calc/subtract/?nums=50,25,12,-7
  • /calc/multiply/?nums=5,5,4
  • /calc/divide/?nums=120,-10,3

Redirect (Golang microservice)

There is only one endpoint which redirects you to the specified target host and path. You can also select protocol by yourself (one from the allowed protocols, which are HTTP, HTTPS, FTP) or go with the default protocol HTTP.

GET /redir/

Redirects you to the specified target in query (and protocol in query).

Examples:

  • /redir/?target=noltio.com&protocol=https
  • /redir/?target=jansvabik.cz
  • /redir/?target=example.com&protocol=ftp
